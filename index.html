Book cover templates in LaTeX with the bookcover document class

The bookcover LaTeX document class can be used to create dust jackets and book covers of hardcover and paperback books (see in CTAN).

If you would like to share your custom book cover template, you can submit it to the email address in the user guide of the bookcover document class.
